# Bicycle Runner

Простой 2D раннер на Canvas с поддержкой мобильного и десктопного управления.

## Возможности

* Горизонтальный уровень (слева направо)
* Камера следует за персонажем
* Полноэкранный режим на мобильных устройствах
* Управление:

  * ПК: стрелки или A/D, прыжок — Space
  * Мобильные устройства:

    * Левая зона — движение
    * Правая зона — прыжок
* Скролл страницы отключён
* Масштабирование Canvas под размер экрана

## Запуск локально

Не рекомендуется открывать `index.html` напрямую через file://

Используйте локальный сервер:

### Вариант 1 — через VS Code

Установите расширение Live Server и выберите "Open with Live Server"

### Вариант 2 — через Python

```
python -m http.server 8000
```

После этого откройте:

```
http://localhost:8000
```

## Публикация через GitHub Pages

1. Залейте проект в репозиторий
2. В настройках репозитория откройте раздел "Pages"
3. Укажите ветку `main` и корневую папку
4. После публикации игра будет доступна по ссылке GitHub Pages

## Технические детали

* Canvas базового размера: 1280x720
* Масштаб рассчитывается динамически
* Камера реализована через `ctx.translate`
* Мобильное управление использует touch events
* Скролл страницы отключён через CSS (`overflow: hidden`, `touch-action: none`)
