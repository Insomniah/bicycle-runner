# Project.md – актуальное состояние после рефакторинга

## Движок и технологии
- Чистый JavaScript (ES6+) + HTML5 Canvas, без фреймворков.
- **Модульная архитектура** – вся логика разбита на ES6 модули (import/export).
- Адаптация под мобильные устройства (тач-зоны, viewport, ориентация).
- Предзагрузка ресурсов через `AssetManager`.
- Ограничение dt ≤ 0.033 сек для стабильности физики.
- Централизованная конфигурация в `config.js` (глобальный объект `CONFIG`).

## Игровой процесс
2D платформер-раннер. Игрок управляет велосипедистом, собирает колёса и достигает правого края уровня.  
После завершения уровня автоматически загружается следующий (level1 → level2).  
Падение вниз → Game Over с рестартом.

## Ключевые особенности

### Модульная архитектура
- Все файлы (кроме `config.js`) – ES6 модули с `export`/`import`.
- Точка входа – `js/main-module.js` (подключается как `<script type="module">`).
- Глобальный объект `window.game` сохранён для совместимости, но вся логика внутри модулей.

### AssetManager (менеджер ресурсов)
- Единый модуль для загрузки и доступа ко всем изображениям.
- Метод `loadAllAssets()` загружает спрайты игрока, фонов, камней, шин, колёс.
- Обеспечивает обратную совместимость (записывает в `window.wheelSprite`, `window.preloadedBackgrounds` и т.д.).
- Упрощает добавление новых ресурсов.

### Динамическая смена фона
- `background.js` умеет загружать разные текстуры (горы, заводы) и масштабировать их (`CONFIG.BACKGROUND_SCALE`).
- Уровни содержат поле `backgroundImage` (например, `CONFIG.BACKGROUND_FACTORIES` для level2).
- В `world.setLevel()` фон переключается автоматически.

### Декорации земли (камни / шины)
- `decorations.js` содержит `rockTypes` (камни) и `tireTypes` (шины).
- На втором уровне (`level.number === 2`) используются шины с масштабом 0.2 и увеличенным количеством.

### Цвет неба
- Уровни могут определять `skyColor`. По умолчанию – `CONFIG.DEFAULT_SKY_COLOR` (чёрный).

### Управление и отладка
- Клавиатура (стрелки/A/D, пробел) + тач-зоны (левая – движение, правая – прыжок).
- Клавиша `ё` включает отладочный режим: хитбокс игрока, координаты, Coyote Time.

## Запуск
1. Убедиться, что в `index.html` подключены только `config.js` и `main-module.js` (как модуль).
2. Запустить Live Server (или любой HTTP-сервер).
3. Игра работает полностью в модульном режиме.
