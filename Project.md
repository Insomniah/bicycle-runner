# Project.md – финальное состояние после полного рефакторинга

## Движок и технологии
- Чистый JavaScript (ES6+) + HTML5 Canvas.
- **Модульная архитектура** – все файлы (кроме `config.js`) – ES6 модули.
- **Конечный автомат** – управление состояниями игры.
- **Событийная шина (Event Bus)** – слабосвязанная коммуникация.
- **AssetManager** – централизованная загрузка ресурсов.
- **Чистые функции** – физика, генерация, расчёты вынесены в отдельные функции.
- **Внедрение зависимостей** – глобальное состояние вынесено в `gameStore`.
- **Уровни в JSON** – динамическая загрузка, легко добавлять новые уровни.
- **Унификация анимаций** – единый класс `SpriteAnimator` для игрока и колеса.
- **Выборочное тестирование** – тесты для коллизий, генерации колёс, аниматора.

## Запуск
- `index.html` подключает `config.js` и `main-module.js` (как модуль).
- Запустить Live Server.
- Уровни загружаются динамически из JSON.
