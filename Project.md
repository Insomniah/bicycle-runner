## Движок и язык
Чистый JavaScript (ES6) + HTML5 Canvas. Без фреймворков.  
Адаптация под мобильные устройства, предзагрузка ресурсов, ограничение dt ≤ 0.033 сек.  
Централизованная конфигурация в `js/config.js`.

## Игровой процесс
2D платформер-раннер. Игрок управляет велосипедистом, собирает колёса и достигает правого края уровня.  
При касании правой границы уровень завершается, игрок автоматически движется вправо, через 2 секунды загружается следующий уровень (level1 → level2).  
Падение вниз → Game Over с возможностью рестарта.

## Ключевые особенности

### Модульная архитектура
- Все файлы (кроме `config.js`) являются ES6 модулями с `export`/`import`.
- Точка входа – `js/main-module.js` (подключается как `<script type="module">`).
- Глобальный объект `window.game` сохраняется для обратной совместимости.

### Динамический фон
- `background.js` умеет загружать разные изображения (`setImage`/`load`) и масштабировать их (`CONFIG.BACKGROUND_SCALE`).
- Уровни содержат поле `backgroundImage` (например, `CONFIG.BACKGROUND_FACTORIES` для level2).
- В `world.setLevel()` фон переключается автоматически.

### Декорации (камни / шины)
- `decorations.js` содержит два набора изображений: `rockTypes` (камни) и `tireTypes` (шины).
- На первом уровне (`level.number === 1`) используются камни, на втором – шины.
- Шины отрисовываются с масштабом 0.2 и в большем количестве.

### Цвет неба
- Уровни могут определять `skyColor`. Если не задан, используется `CONFIG.DEFAULT_SKY_COLOR` (чёрный).

### Управление и отладка
- Клавиатура (стрелки/A/D, пробел) + тач-зоны.
- Клавиша `ё` переключает отладочный режим (хитбокс игрока, координаты, Coyote Time).
- Отладка реализована в `window.game.drawDebug()`.

### Исправленные баги
- **Рестарт колёс** – в `window.game.restart()` добавлен вызов `world.generateWheels()`.
- **Смена фона** – объект `mountains` переименован в `background`, папка `background` переименована в `scenery`.
- **Переход на модули** – все зависимости явно импортируются, ошибок с `canvas is not defined` больше нет.

## Запуск
1. Убедитесь, что в `index.html` подключены только `config.js` и `main-module.js` как модуль.
2. Запустите Live Server (или любой HTTP-сервер).
3. Игра готова к работе.
