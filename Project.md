# Project.md – актуальное состояние (после исправления путей для GitHub Pages)

## Движок и технологии
- Чистый JavaScript (ES6+) + HTML5 Canvas.
- **Модульная архитектура** – ES6 modules.
- **Конечный автомат**, **Event Bus**, **AssetManager**, **чистые функции**, **внедрение зависимостей**.
- **Уровни в JSON**, **унификация анимаций**, **выборочное тестирование**.
- **Современное мобильное управление** – виртуальный джойстик и кнопка прыжка.

## Игровой процесс
2D платформер-раннер. Игрок собирает колёса, достигает правого края → следующий уровень. Падение → Game Over с рестартом. Управление: клавиатура (для ПК) или сенсорный джойстик + кнопка (для мобильных).
