# Project.md – актуальное состояние (рефакторинг завершён)

## Движок и технологии
- Чистый JavaScript (ES6+) + HTML5 Canvas, без фреймворков.
- **Модульная архитектура** – вся логика разбита на ES6 модули (import/export).
- **Событийная шина (Event Bus)** – связь между модулями через события.
- **AssetManager** – централизованная загрузка ресурсов.
- **Чистые функции** – физика, генерация декораций, расчёт камеры, облаков, фона вынесены в отдельные функции без побочных эффектов.
- Адаптация под мобильные устройства, предзагрузка ресурсов, ограничение dt.
- Централизованная конфигурация в `config.js`.

## Игровой процесс
2D платформер-раннер. Игрок собирает колёса, достигает правого края → следующий уровень. Падение → Game Over с рестартом.

## Архитектурные решения

### Модули ES6
- Все файлы (кроме `config.js`) – модули с `export`/`import`.
- Точка входа – `main-module.js` (подключается как `<script type="module">`).

### Event Bus
- Модуль `core/eventBus.js` с методами `on`, `off`, `emit`.
- Используется для событий: `level.complete`, `game.over`, `wheel.collected`.
- Игровой цикл не содержит логики проверки `state.gameOver` – всё через события.

### Чистые функции (Pure Functions)
- **`player.js`** – `updatePlayerPhysics` вычисляет новое состояние игрока на основе входных параметров.
- **`world.js`** – `generateWheelsFromData`, `resolveBackgroundImage` – создание колёс и выбор фона без мутаций.
- **`camera.js`** – `calculateCameraPosition` – расчёт позиции камеры.
- **`controls.js`** – `processTouches` – обработка касаний.
- **`decorations.js`** – `generateDecorationsList` – генерация камней/шин.
- **`background.js`** – `calculateBackgroundTiles` – расчёт тайлов фона.
- **`sky.js`** – `generateClouds`, `updateClouds`, `getCloudDrawParams` – работа с облаками.

### AssetManager
- Загружает и кеширует все изображения.
- Предоставляет доступ через `assetManager.get(key)`.
- Сохраняет обратную совместимость (записывает в `window` там, где это ещё нужно).

## Ключевые особенности

### Динамический фон
- `background.js` загружает разные текстуры (горы, заводы) и масштабирует их.
- Уровни содержат поле `backgroundImage`, которое используется в `world.setLevel()`.

### Декорации (камни / шины)
- На втором уровне (`level.number === 2`) отрисовываются шины (масштаб 0.2, больше объектов).
- На первом – камни.

### Цвет неба
- Уровни могут определять `skyColor`. По умолчанию – `CONFIG.DEFAULT_SKY_COLOR` (чёрный).

### Управление и отладка
- Клавиатура (стрелки/A/D, пробел) + тач-зоны.
- Клавиша `ё` включает хитбокс и отладочную информацию.
